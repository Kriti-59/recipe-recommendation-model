
# 🍳 Recipe Recommendation Model  

## 📖 Overview  
Cooking with limited ingredients is a common challenge for students (and honestly, anyone with a half-empty fridge). Instead of eating out or stressing over what to cook, this project helps you find recipes based on what you already have at home.  

Our model takes in a list of ingredients from the user and recommends recipes with step-by-step instructions for preparing a homemade meal.
